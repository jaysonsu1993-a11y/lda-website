# 竞品调研

## 2026-03-19

### Morningstar Engineering (morningstar.io / shop.morningstar.io)

**行业标杆，定价参考：MC4 PRO $399 / MC8 PRO $599**

---

#### 首页设计

- **极简风格**：大量留白，字体主导，无多余装饰
- **导航极简**：Store / Products / Support / More（4项）
- **Hero 区**：大字体产品名 + 一句话价值主张（如 "Compact without compromise"）
- **CTA 风格**：纯文本链接（非按钮），蓝色强调色，邀请式而非销售式

#### 商店页设计

- **产品卡片**：白色边框盒子，高对比度产品图，标题→评分→价格
- **无直接购买按钮**：网格页不显示 Add to Cart，引导进入详情页
- **货币显示**：主货币（USD）粗体，副货币（HKD 换算）小字
- **社交证明**：评分 + 评论数直接显示在标题下方
- **库存提示**：产品卡片角标显示 "Early April Restock"

#### 产品详情页

- 404 错误（可能是区域路由问题）

#### 导航结构

- 主站（morningstar.io）和商店（shop.morningstar.io）**分离**
- 顶部 Banner：Free shipping 激励
- 分类导航：文字链接居中
- 右侧悬浮：搜索图标、购物袋、货币选择器、返回主站

#### 设计语言

- **黑白灰单色 + 蓝色强调**
- **字体**：几何无衬线，重量对比强烈（标题粗体 + 描述细体）
- **Bento Box 布局**：不同背景色分隔不同产品线
- **文本链接 CTA**：高端品牌风格（参考 Apple、Teenage Engineering）
- **FAQ 手风琴**：极简无边框，纯横线分隔

#### 功能亮点

- **Shop by Category**：图片分类导航
- **Trusted by pro musicians**：社交证明区块
- **FAQ 图标**：地球（配送）、扳手（设置），快速扫描

---

### 值得 LdA 参考的点

1. **文本链接 CTA** — 极简品牌感强，邀请式引导
2. **双货币显示** — 国际用户友好（可考虑 USD + CNY）
3. **主站 + 商店分离** — 品牌站轻量、商店独立
4. **产品网格页无直接购买按钮** — 高单价产品引导深度阅读
5. **极简 FAQ 手风琴** — 无边框风格节省空间
